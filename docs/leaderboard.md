The leaderboard reports the quality performance of each model across the evaluation dataset. You can check models' setup in [examples](https://github.com/ITMO-NSS-team/Aiice/tree/main/scripts/experiments).

!!! note
    In order to add your model to the leaderboard, see [leaderboard instruction](https://github.com/ITMO-NSS-team/Aiice/tree/main/scripts/experiments/README.md) and [Contributing](contributing.md) doc.


All metric definitions, formulas, and their interpretations are described in detail on the [API Reference - Metrics](api/metrics.md).

!!! note
    Models are ranked best → worst for each metric. Bar lengths are min–max normalized across all models in the current view — the best model always gets the fullest bar (100%). Radar chart values are normalized the same way: closer to the edge means better.
